# Practice Questions

This section contains a comprehensive list of practice questions for MongoDB Aggregation, categorized by difficulty level. These exercises are designed to help you master aggregation pipelines step-by-step.

The practice questions are divided into three levels:

### 1. **[Beginner Level](./03-beginner-aggregation.md)**

- **Focus**: Fundamental building blocks.
- **Concepts**:
  - Filtering (`$match`) & Sorting (`$sort`)
  - Grouping (`$group`) & Accumulators (`$sum`, `$avg`)
  - Basic Joins (`$lookup`)
  - Array Unwinding (`$unwind`)
  - Projection (`$project`)

### 2. **[Intermediate Level](./04-intermediate-aggregation.md)**

- **Focus**: Multi-stage transformations and analytical grouping.
- **Concepts**:
  - Post-grouping filtering (`$match` after `$group`)
  - Data Bucketing (`$bucket`)
  - Multi-faceted Aggregation (`$facet`)
  - Computed Fields (`$addFields`)
  - Array Filtering & Mapping (`$filter`, `$map`)

### 3. **[Advanced Level](./05-advanced-aggregation.md)**

- **Focus**: Complex business logic and advanced analytics.
- **Concepts**:
  - Pipeline Lookups (Correlated Subqueries)
  - Window Functions (`$setWindowFields`, `$rank`)
  - Union Queries (`$unionWith`)
  - Conditional Logic (`$cond`, `$switch`)
  - Root Document Manipulation (`$replaceRoot`, `$mergeObjects`)
